# Voided Mod Loader

A modern **mod loader** and mod management dashboard built with Astro, Shadcn UI, and Cloudflare.

Browse, install, and manage mods for your favorite games. Deployable instantly to Cloudflare Workers.

## Features
- 🎮 Mod browser with search & categories
- 📦 One-click install / download
- ✅ Manage installed mods (enable/disable/update)
- 👤 Game profiles / instances
- 🔐 API for external tools
- 🚀 Cloudflare-powered (fast & cheap)
